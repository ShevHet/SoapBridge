# IcutechTestApi.Tests

Unit-тесты для проекта IcutechTestApi.

## Технологии

- **xUnit** - фреймворк для тестирования
- **Moq** - библиотека для создания моков
- **FluentAssertions** - библиотека для более читаемых утверждений

## Запуск тестов

```bash
dotnet test
```

## Структура тестов

### AuthControllerTests

Тесты для контроллера аутентификации покрывают следующие сценарии:

#### Login Tests
- ✅ Успешный вход с валидными учетными данными (200 OK)
- ✅ Неудачный вход с неверными учетными данными (401 Unauthorized)
- ✅ Ошибка SOAP-клиента (500 Internal Server Error)
- ✅ Ошибка валидации параметров (400 Bad Request)
- ✅ Неожиданная ошибка (500 Internal Server Error)

#### Register Tests
- ✅ Успешная регистрация с валидными данными (200 OK)
- ✅ Неудачная регистрация (400 Bad Request)
- ✅ Ошибка SOAP-клиента (500 Internal Server Error)
- ✅ Ошибка валидации параметров (400 Bad Request)
- ✅ Неожиданная ошибка (500 Internal Server Error)
- ✅ Регистрация с опциональными полями (null значения)

## Покрытие

Все основные сценарии контроллера покрыты unit-тестами с использованием моков для `ISoapAuthClient`.
