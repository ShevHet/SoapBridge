## Цветовая палитра

- **Primary Color**: #6366f1 (Indigo)
- **Primary Hover**: #4f46e5 (Darker Indigo)
- **Success**: #10b981 (Green)
- **Error**: #ef4444 (Red)
- **Warning**: #f59e0b (Amber)

## Типографика

- **Font Stack**: System fonts (-apple-system, BlinkMacSystemFont, Segoe UI, Roboto)
- **Заголовок**: 2rem, font-weight: 700
- **Подзаголовок**: 0.95rem, opacity: 0.9
- **Текст формы**: 1rem, line-height: 1.6

## Компоненты

### Карточка
- Белый фон (#ffffff)
- Скругление: 1rem (16px)
- Тень: 0 20px 25px -5px rgba(0, 0, 0, 0.1)
- Hover эффект: увеличенная тень

### Input поля
- Высота: 48px (крупные)
- Padding: 1rem 1.5rem
- Border: 2px solid #e2e8f0
- Focus: синяя рамка + тень
- Скругление: 0.5rem

### Кнопка Primary
- Фон: #6366f1
- Hover: #4f46e5 + поднятие на 1px
- Тень при hover
- Плавные переходы

### Адаптивность
- Mobile: 90% ширины экрана
- Desktop: 400-600px (центрировано)
